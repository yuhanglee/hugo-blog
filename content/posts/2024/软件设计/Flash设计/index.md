Flash设计需要考虑当前数据存储和扇区大小，尽量保证每个扇区都尽可能的被使用到，以防浪费。

需要在Flash中存放的内容按照状态可以分为:

| 分区名称     | 是否被修改 | 地址是否固定 |
| ------------ | ---------- | ------------ |
| Boot代码     | NO         | YES          |
| Boot参数     | YES        | YES          |
| 应用代码     | NO         | NO           |
| 参数区       | YES        | NS           |
| 版本信息     | NO         | NO           |
| 硬件配置信息 | NO         | NO           |
|              |            |              |
